# WaveFront Obj File Converter

Convert to:

- gltf/glb
- 3d tiles(b3dm) with batch ids

Obj file limitations:

1. one mtl file
2. all faces should be triangles already
3. only convert ka, kd, map_ka, map_kd, Ns, d material info
